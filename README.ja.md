# YouTube Helper

YouTube用のChrome拡張機能。

## 使い方

1. [Releasesページ](../../releases) から `youtube-helper.zip` をダウンロードし、解凍します。
2. `chrome://extensions` にアクセスし、拡張機能の管理ページを開きます。
3. 右上の **デベロッパー モード** をオンにします。
4. **パッケージ化されていない拡張機能を読み込む** ボタンをクリックし、解凍したフォルダを選択します。
